# Manual Manual

```
git clone https://github.com/jwestall/astronome.git
cd astronome
sudo just install
```

# Build
```
git clone https://github.com/jwestall/astronome.git
cd astronome
cargo build
```

## Dependencies
- [libcosmic](https://github.com/pop-os/libcosmic?tab=readme-ov-file#building)
